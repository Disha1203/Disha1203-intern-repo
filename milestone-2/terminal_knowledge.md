## Goal

A good terminal setup makes development smoother. Choose a terminal client and get comfortable using it.

---

## Relections

### Which terminal client did you choose? Why?

* I chose the VS Code integrated terminal using zsh on macOS. 
* Since I already use VS Code for development, having the terminal inside the editor makes it convenient to run commands, manage Git repositories, and execute programs without switching applications.

### What customizations (if any) did you make?

* Installed and configured the zsh-autosuggestions and zsh-syntax-highlighting plugins to improve the terminal experience.
* zsh-autosuggestions provides command suggestions based on command history
* While zsh-syntax-highlighting highlights valid and invalid commands in real time, improving usability and productivity.

### What was the most useful command you learned today?

`source ~/.zshrc `
*  It lets me load my shell setup right after I've made changes to aliases, plugins or environment variables without having to restart the terminal. 
* This simplified and sped up zsh configuration.

`tree`
* To visualise tree structure

`history`
* Shows previously executed commands
