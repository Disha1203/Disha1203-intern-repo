## Top 10 shortcuts

### Navigation

* `⌃ + ⌥ + Left/Right arrow` : Navigate to the left/right tabs on the editor
* `⌘ + Shift + P` : Command Palette
* `⌃ + Tab` : To switch btw open tabs

### Editing 

* `⌘+D` : Select Multiple Occurrences Of The Same Word
* `⌘+/` : Toggle line comment
* `⌥ + Click` : Multiple cursors
* `⌥ + ⌘ + ↑ / ↓` : Insert cursor above/below

### Debugging

* `F5` : Start/Continue Debugging
* `Shift + F5` : Stop Debugging
* `F10` : Step Over
* `F11` : Step Into
### Terminal 

* `` ⌃ + `` : To show integrated terminal
* `↑` : Previous Command
* `⌘ + K` : Clear Terminal
